01. 1부터 100까지 출력하기(for문, while, do while)

for문은 변수 i를 1로 초기화하고, i가 100보다 작을 동안(i < 100) 반복합니다. 각 반복에서 i를 출력하고, 반복이 진행될 때마다 i를 1씩 증가시킵니다. while문은 num 변수를 1로 초기화하고, num이 100보다 작을 동안(num < 100) 반복합니다. 각 반복에서 num을 출력하고, 반복이 진행될 때마다 num을 1씩 증가시킵니다. do while문은 먼저 루프 본문을 실행하고, 나중에 조건을 검사합니다. 따라서, num을 출력하고 나서 num을 증가시키며, 그 후에 num이 100보다 작은지 확인하여 조건을 만족할 때까지 반복됩니다. 


    {
        for (let i = 1; i < 100; i++) {
            console.log(i);
        }
    }
    {
        let num = 1; // 초깃값;

        while (num < 100) {  // 조건식;
            console.log(num); // 실행문;
            num++; // 증감값;
        }
    }
    {
        let num = 1; // 초깃값;

        do {
            console.log(num);  // 실행문;
            num++;  // 증감값;
        } while (num < 100);  //조건식;
    }
결과 확인하기
1....100

02. 1부터 100까지 짝수만 출력하기(for문, if문)

for문을 사용하여 1부터 100까지의 숫자를 반복합니다. 각 숫자가 짝수인지를 판별하기 위해 if문을 사용합니다. 짝수인 경우에만 해당 숫자를 출력합니다. % 연산자는 나머지를 반환하는 연산자입니다. 따라서 i % 2 == 0는 i를 2로 나눈 나머지가 0인지를 확인하여 짝수인지를 판별합니다. 짝수인 경우에만 해당 숫자를 출력합니다. 


{
    for (let i = 1; i <= 100; i++) {
        if (i % 2 == 0) {
            console.log(i);
        }
    }
}
결과 확인하기
2,4,6....100

03. 1부터 100까지 짝수는 파란색, 홀수는 빨간색 출력하기

주어진 코드는 HTML과 JavaScript를 사용하여 1부터 100까지의 숫자를 출력하는데, 짝수는 파란색으로, 홀수는 빨간색으로 표시하는 코드입니다. 


{
    for (i = 1; i <= 100; i++) {
        if (i % 2 == 0) {
            document.write("" + i + "");
        } else {
            document.write("" + i + "");
        }
    }
}
결과 확인하기
1부터100까지 짝수는 파란색, 홀수는 빨간색

04. 중첩 for문을 이용해서 구구단 출력하기

외부 for문에서 2부터 9까지의 숫자를 반복하고, 내부 for문에서는 1부터 9까지의 숫자를 반복하여 각 단의 곱셈 결과를 출력합니다. 외부 for문이 2일 때 내부 for문이 실행되며 2단의 곱셈 결과가 출력됩니다. 이런 식으로 2부터 9까지의 숫자에 대해 각각의 단을 출력합니다. 


{
    for (let i = 2; i <= 9; i++) {
        for (let j = 1; j <= 9; j++) {
            console.log(i + "*" + j + "=" + i * j);
        }
    }
}
결과 확인하기
2*1=2
2*2=4
2*3=6
...
9*8=72
9*9=81

05. 중첩 for문을 이용해서 8단 출력하기

외부 for문에서 8부터 8까지의 숫자를 반복하고, 내부 for문에서는 1부터 9까지의 숫자를 반복하여 8단의 곱셈 결과를 출력합니다. 외부 for문의 조건이 i <= 8로 되어 있으므로 8단만 출력됩니다. 내부 for문은 1부터 9까지 반복되어야 하는데, 코드상에는 1부터 8까지만 반복하도록 되어 있습니다. 이는 내부 for문의 조건이 j <=8로 되어 있기 때문입니다. 따라서 실행 결과는 8단의 곱셈 결과만 출력됩니다. 


{
    for (let i = 8; i <= 8; i++) {
        for (let j = 1; j <= 8; j++) {
            console.log(i + "*" + j + "=" + i * j);
        }
    }
}
결과 확인하기
8*1=8
8*2=16
8*3=24
8*4=32
8*5=40
8*6=48
8*7=56
8*8=64
8*9=72
